Actors

    GameDetail (originally - GameDetails)

    A list of details associated with the game.
    Each GameDetail may have zero or more associated Games
    CRUDNameTypeOriginal Name
    GameDetailId     		string
    Game parent
    Name
    The Name of an attribute of the project
    		string
    Value
    The Value of an attribute of the project
    		string
    GameLoadingScreen
    The name of the game associated with the game detail.
    		string
    Game
    The game that this attribute/detail is associated with.
    		string
    GameIsActive
    A flag indicating if the game is the active game.
    		boolean
    GameIsDisabled
    A flag indicating if the associated game has been disabled (removed).
    		boolean
    GameDetail Query
    SELECT GameDetailId, Name, Value, GameLoadingScreen, Game, GameIsActive, GameIsDisabled FROM GameDetail

    Character (originally - Characters)

    A list of characters in the game.
    Each Character may have zero or more associated Games
    CRUDNameTypeOriginal Name
    CharacterId     		string
    Game parent
    Name
    Name of the character.
    		string
    Avatar
    A url to an image for this character in the game
    		string
    Type
    The type of character (Marshmallow, Ninja, NPC, etc.)
    		string
    Description
    Short description of the character for the game.
    		string
    IsMissing
    A flag indicating if this character is 'missing' - i.e. they have not been found yet.
    		boolean
    IntroducedAtLevel
    The number of the level which the character shows up. For some games, all characters will be introduced right out of the gate. This field is agnostic of the specific values - and it can be left blank.
    		int32
    Game
    The game associated with this character.
    		string
    GameIsActive
    A flag for whether the game associated with this character is active.
    		boolean
    Character Query
    SELECT CharacterId, Name, Avatar, Type, Description, IsMissing, IntroducedAtLevel, Game, GameIsActive FROM Character

    AppUser (originally - AppUsers)

    A list of users allowed to access games.
    CRUDNameTypeOriginal Name
    AppUserId     		string
    Name
    Name of the app user.
    		string
    EmailAddress
    The email address associated with the app user.
    		string
    Roles
    Roles associated with the games.
    		string
    Notes
    Notes about the app user.
    		string
    AppUser Query
    SELECT AppUserId, Name, EmailAddress, Roles, Notes FROM AppUser

    Game (originally - Games)

    A list of games defined.
    Each Game may have zero or more associated GameDetails , Characters
    CRUDNameTypeOriginal Name
    GameId     		string
    GameDetails parent
    Characters parent
    Name
    Name of the character.
    		string
    LoadingScreen
    Loading screen image.
    		string
    Notes
    Notes about the game
    		string
    IsActive
    A flag indicating if this game is the active game.
    		boolean
    GameDetails
    Links to the details about the game
    		string
    Characters
    A list of characters in the game.
    		string
    IsDisabled
    Flag indicating if the game is disabled.
    		boolean
    Game Query
    SELECT GameId, Name, LoadingScreen, Notes, IsActive, GameDetails, Characters, IsDisabled FROM Game
